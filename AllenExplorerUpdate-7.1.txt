Allen Explorer v7.1 Update Log
---------------------------------------------
New: Support for multi-monitor extended mode.
New: Adaptive filename display in thumbnail mode.
New: Significantly improved browsing speed for folders with many images.
New: Thumbnail Huge (320x320), Thumbnail Extra Huge (640x640).
New: Hotkeys Ctrl+Shift+Tab, Ctrl+PageDown, Ctrl+PageUp.
New: Hide window when dragging file items.
New: Tab footprint tracking.
New: Centered display option for status bar information.
New: Appearance – Custom themes.
New: Appearance – Set folder colors.
New: Color marking for folders upgraded to colorful folders.
New: Operation bar buttons – Copy name, Copy path.
New: Record and display access count, support sorting by access count.
New: Record and display last access time, support sorting and grouping by last access time.
Optimized: Automatically update favorites, frequent folders, and recent items after file deletion, move, or rename.
Optimized: Retain file marks and visit records after restoring from Recycle Bin.
Optimized: Sync colors, progress, tags, and notes when opening the same folder in multiple windows or workspaces.
Fixed: Issue where some fixed system menu items could not be opened.
Others: Performance and stability optimizations.
